# Backend - Django (Python)

### Primeiros passos
- Criar um ambiente virtual
  - **Windows**
    ```shell
    py -m venv .venv
    ```
    ```shell
    ./.venv/Scripts/activate
    ```
- Instalando Django
  - **Windows**
    ```shell
    pip install django
    ```
- Inicando um projeto Django
  - **Windows**
    ```shell
    django-admin startproject taskflow .
    ```
